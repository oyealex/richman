# 🎲 终端大富翁 — 游戏规则说明书

---

## 欢迎来到大富翁！

这是一款回合制策略桌游，适合 2~4 人游玩。你的目标是成为**最后一个没有破产的玩家**——通过购买地块、升级房屋、收取租金，把其他玩家统统逼到破产！

---

## 一、游戏概览

| 项目 | 说明 |
|---|---|
| 玩家人数 | 2~4 人（可含 AI） |
| 游戏类型 | 回合制，按顺序轮流行动 |
| 核心玩法 | 掷骰子走格子 → 买地 → 踩中自己的地时升级 → 收租 |
| 胜利条件 | 其他所有玩家破产，你是最后一个存活者 |
| 游戏时长 | 约 30~60 分钟（视棋盘大小和人数而定） |

### 游戏准备

| 项目 | 说明 |
|---|---|
| 起始现金 | 每位玩家 $2000 |
| 起始位置 | 所有玩家从起点格出发，**游戏开始时位于起点不触发奖金** |
| 行动顺序 | 游戏开始时随机确定，之后固定循环 |

---

## 二、你能看到什么？——信息可见性

在大富翁中，有些信息是公开的（所有玩家都能看到），有些则是私密的（只有你自己知道）。了解什么信息可见、什么不可见，是制定策略的关键。

### 公开信息（所有玩家可见）

| 信息类别 | 具体内容 |
|---|---|
| 棋盘地图 | 所有格子的类型、位置；每块地块归属谁；已购地块的当前等级（空地/1级/2级/3级） |
| 玩家位置 | 每位玩家的棋子当前所在格子 |
| 玩家状态 | 哪位玩家在监狱中、剩余关押轮数 |
| 回合信息 | 当前第几回合、当前正在行动的玩家是谁 |
| 骰子点数 | 当前行动玩家掷出的点数 |
| 机会卡公示 | 任何玩家踩中机会卡格后，**抽到的卡牌内容立即公开**；卡牌效果是否触发也公示 |
| 卡牌使用公示 | 任何玩家主动使用保留类卡牌（免狱卡、拆除卡）时，**使用者、目标、结果全部公开** |
| 破产公示 | 任何玩家破产时，全场公开 |

### 私密信息（仅自己可见）

| 信息类别 | 具体内容 |
|---|---|
| 现金数额 | 只有你自己知道你有多少钱 |
| 手牌 | 你持有的免狱卡、拆除卡数量，只有你知道 |
| 资产投入 | 虽然别人知道哪些地是你的以及地块等级，但他们不知道你为每块地具体投入了多少钱 |

## 三、棋盘是什么样子的？

棋盘是一个**环形路径**，大约有 30 个格子围成一圈。所有玩家从同一个起点出发，顺时针不断绕圈行走。

棋盘上的格子分为六种：

### 🏁 起点格
- 棋盘上的第 1 格，也是绕完一圈后回到的地方
- **游戏开始时所有玩家位于起点，不触发奖金**
- 之后**每当经过或停留在起点**，银行会发给你一笔固定奖金（例如 $200）
- 这是你最重要的稳定收入来源！

### 🏠 地块格
- 这是棋盘上数量最多的格子，每一格都代表一块可以购买的土地
- 踩上空地可以买下来，以后别人踩上来就要给你交租金
- 买下来的地还可以花钱升级，升级后租金更高
- **注意：只有当你踩中自己拥有的地块时，才能对它进行升级，且每次踩中只能升一级**

### 🎴 机会卡格
- 踩中这个格子，你会从机会卡堆里**随机抽取一张**
- 卡片效果有好有坏——可能是天降横财，也可能是飞来横祸
- 免狱卡和拆除卡可以保留在手中，其余卡片立即生效
- **抽到的卡片内容会向所有玩家公开**

### 🚔 入狱格
- 踩中这个格子，你会被直接送进监狱
- **如果你手中有免狱卡，可以在此刻使用，避免入狱**

### 🔒 监狱格
- 被关押的玩家所在的位置
- 你的棋子会暂时待在这里，直到关押期满

### ⬜ 空白格
- 什么也没有的格子
- 踩上去无事发生，安心路过即可

## 四、一个回合里你能做什么？

每个玩家的回合严格按照以下步骤执行：

```
您的回合开始
    │
    ▼
① 效果更新阶段
   更新你身上所有持续中的效果
   （例如：监狱剩余关押轮数 -1，未来可能还有
    其他持续效果在此阶段更新）
   然后按效果获得的先后顺序依次触发效果
   （例如：如果倒计时归零，触发"强制释放"出狱）
    │
    ▼
② 掷骰移动阶段
   掷一颗骰子（1~6），按点数顺时针前进
   移动过程中经过起点时领取奖金
    │
    ▼
③ 落点效果阶段
   根据你最终停下的格子类型触发效果：
   · 起点格：确认起点奖金已结算；如果是本次移动正好停在起点，不会重复领取
   · 空地格：无事发生（购买在阶段④）
   · 他人地块：支付租金
   · 自己的地块：无事发生（升级在阶段④）
   · 机会卡格：随机抽取一张机会卡（内容公开）
   · 入狱格：触发入狱（如有免狱卡可在此时使用避免入狱）
   · 空白格：无事发生

   ⚠️ 重要：如果机会卡抽到"移动卡"（前进/后退），
      移动后对新落点按阶段③规则重新处理。
      这意味着移动卡可能引发连锁落点效果
      （例如移动后又踩中机会卡格，再抽一张）。
      连锁过程中只处理落点效果；购买、升级、使用拆除卡等
      阶段④动作，只会在所有连锁结束后的最终落点执行一次。
    │
    ▼
④ 玩家动作阶段
   根据你当前停在的格子类型，可执行的动作如下：

   | 你停在 | 阶段④可做的事 |
   |---|---|
   | 空地块 | 购买该地块（支付地价），或不买 |
   | 自己的地块 | 升级该地块一级（支付升级费），或不升级 |
   | 任意格子（含上述两种） | 使用手中的拆除卡 |
   | 在监狱中 | 无阶段④（关押中不能行动） |
    │
    ▼
⑤ 回合结束，轮到下一位玩家
```

> 💡 注意：阶段④中，如果你停在自己的地块上，**只能升一级，只能升一次**。下次再踩到同一块地才能继续升级。

简单来说就是：**更新效果 → 掷骰 → 走 → 落点效果 → 动作 → 下一位**。

## 五、移动规则详解

### 5.1 怎么走？
- 每回合掷**一颗六面骰子**，掷出的数字（1~6）就是你要走的步数
- 方向永远是顺时针，走完一圈回到起点继续

### 5.2 经过起点
- 在你移动的过程中，只要**路过或正好停在**起点格，就能领奖金
- 奖金金额是固定的（比如 $200），由银行自动发放
- **注意：游戏开始时所有玩家位于起点，此初始位置不触发奖金**
- 如果一回合内经过起点多次（比如抽到"前进到起点"的卡），每次经过都发奖金

### 5.3 经过他人地块
- 移动过程中**仅仅经过**（路过但不停留）他人拥有的地块时，**无需支付租金**
- 只有**最终停在那块地上**才需要付租金

## 六、地块——你的核心资产

### 6.1 每块地长什么样？

每块地都有以下属性，你在棋盘上可以看到：

| 属性 | 说明 | 举例 |
|---|---|---|
| 名称 | 地块的名字 | "海滨别墅" |
| 地价 | 买下这块空地需要多少钱 | $120 |
| 租金（空地） | 没升级时，别人踩上来要付多少钱 | $8 |
| 租金（1级） | 升到 1 级后的租金 | $30 |
| 租金（2级） | 升到 2 级后的租金 | $90 |
| 租金（3级） | 升到 3 级后的租金 | $250 |
| 升级费 | 每升一级要花多少钱 | $50 |

### 6.2 买入地块
- 当你踩中一块**还没有主人**的地块时，在阶段④（玩家动作阶段）你可以选择：
  - ✅ **买下它**：支付地价，这块地从此归你所有
  - ❌ **不买**：什么也不发生，地块保持无主状态，以后别人踩到也可以买

> ⚠️ 注意：不买就是真的不买。没有拍卖、没有人能抢——下次谁踩到谁有购买权。

### 6.3 你能同时拥有多少块地？
- **没有上限**。只要你有钱，你可以把整张棋盘买下来。
- 地块之间**互不关联**——没有颜色组，没有"必须凑齐一套"的限制。每一块地都是完全独立的资产。

---

## 七、升级你的地块

### 7.1 什么时候可以升级？
- **只有当你踩中自己已经拥有的地块时**，才能选择对该地块进行升级
- 升级发生在阶段④（玩家动作阶段）
- **每次踩中只能升一级，只能升一次**——不能连升多级
- **不能随意升级自己没有踩中的地块**——你必须亲自走到那块地上才行

### 7.2 怎么升级？
- 每块地有 4 个等级：**空地 → 1级 → 2级 → 3级**
- 每次升级提升一级，支付对应的升级费
- 升级后，地块的租金会大幅上涨

### 7.3 升级有什么限制？
- 最高只能升到 **3 级**
- 每次踩中只能升一级
- **没有其他限制**！只要你踩中了自己的地，就可以升级它，不需要持有相邻地块，不需要均匀发展

### 7.4 升级带来的收益
举个直观的例子：

> 你花 $120 买下"海滨别墅"，此时租金是 $8。
> 后来你再次踩中它，花 $50 升到 1 级 → 租金涨到 $30。
> 又过了几圈再次踩中，花 $50 升到 2 级 → 租金涨到 $90。
> 又一次踩中，花 $50 升到 3 级 → 租金涨到 $250。
>
> 投资总计 $270，但别人踩一脚就要付 $250，踩两次就回本了！

---

## 八、租金——怎么从别人身上赚钱？

### 8.1 什么时候要付租金？
- 当你的棋子**最终停在**别人拥有的地块上时，你必须向地主支付租金
- 仅经过不停留，**不需要付租金**
- 租金金额 = 该地块**当前等级对应的租金**

### 8.2 如果付不起怎么办？
- 你会进入**强制破产处理**（见第十一章），银行会帮你变卖资产来凑钱

### 8.3 特殊情况：地主在监狱里
- 如果某块地的主人正被关在监狱里，这块地**暂时不收租金**
- 别人踩上去不用付钱——这给了其他玩家喘息的机会，也让坐牢的代价变得更加真实

---

## 九、机会卡——命运的骰子之外

### 9.1 什么时候抽卡？
- 在阶段③踩中**机会卡格**时，立刻从卡堆中随机抽取一张
- 卡堆里有多少张卡都可以，不会用完
- **抽到的卡片内容会向所有玩家公开**

### 9.2 卡片有哪些效果？

| 类型 | 生效方式 | 说明 |
|---|---|---|
| 💰 金钱获得 | 立即生效 | 从银行获得一笔钱 |
| 💸 金钱损失 | 立即生效 | 向银行支付一笔钱 |
| 🚶 移动 | 立即生效 | **具体点数在抽到时随机决定**；移动后对新落点按阶段③规则重新处理 |
| 🚔 入狱 | 立即生效 | 直接送入监狱（如有免狱卡可在此刻使用避免） |
| 🛡️ 免狱卡 | 保留手牌，主动使用 | 即将入狱时使用，免于入狱 |
| 🏚️ 拆除卡 | 保留手牌，主动使用 | 保留在手中，在阶段④使用 |

### 9.3 卡片的生效方式
- **立即生效类**（金钱、移动、入狱）：抽到后立刻执行效果，然后丢弃
- **可保留类**（免狱卡、拆除卡）：抽到后放入手牌，由你在合适的时机主动使用

### 9.4 移动卡的详细规则
- 移动卡在抽到时随机决定具体移动点数（前进或后退）
- 按点数移动后，对新落点**按阶段③规则重新处理**
- 可能引发连锁：移动后又踩中机会卡格 → 再抽一张；移动后又踩中入狱格 → 入狱
- 连锁过程中只处理阶段③落点效果，不会在中间落点购买、升级或使用拆除卡；所有连锁结束后，才根据最终落点进入一次阶段④
- 移动过程中经过起点，正常领取奖金

### 9.5 免狱卡
- 保留在手中，无限期持有，不会过期
- **使用时机**：当你即将入狱的那一刻——踩中入狱格、抽到入狱卡、或移动卡连锁导致落入入狱格时，均可在生效前使用免狱卡
- 使用时消耗一张免狱卡，**跳过本次入狱**，继续正常游戏流程
- **不能在已经入狱之后使用**——入狱后只能等待关押期满
- 不可买卖、不可交易
- 手中可同时持有任意多张
- **使用免狱卡时信息公开**：谁使用了卡、避免了哪次入狱，全部公开
- **手牌数量为私密信息，其他玩家不知道你持有几张**

### 9.6 拆除卡
- 保留在手中，无限期持有，不会过期
- 只能在阶段④（玩家动作阶段）使用
- 使用时的规则：
  - 范围：以你所在格为中心，沿棋盘**顺时针和逆时针方向各延伸 12 格**
  - 在环形棋盘上，如果棋盘总格数为 N，实际范围为两个方向上各 min(12, N-1) 格，且不可重复选中同一格
  - 你可以在范围内选择一块**有等级的地块**（无论主人是谁，包括你自己）
  - 选中的地块**降低一级**（3级 → 2级 → 1级 → 空地）
  - 如果目标地块已经是空地（0 级），则无法拆除，需另选目标
  - 如果范围内没有任何可拆除的地块，此卡无法使用
- **使用拆除卡时信息公开**：谁使用了卡、拆了哪块地、从几级拆到几级，全部公开
- 不可买卖、不可交易
- **手牌数量为私密信息，其他玩家不知道你持有几张**

### 9.7 卡片通用规则
- 可保留类卡片不可买卖、不可交易、不会过期
- 立即生效类卡片抽到即生效，用完丢弃
- **主动使用保留类卡片时，使用者、目标、结果全部公开**

## 十、监狱——如何进去，如何出来

监狱是大富翁中最严重的惩罚。一旦入狱，你只能等待关押期满，没有任何捷径可走。

### 10.1 怎么进监狱？（4 种触发场景）

| 场景 | 说明 |
|---|---|
| 🚔 踩中入狱格 | 走到棋盘上的"入狱"格，立即入狱 |
| 🎴 抽到入狱卡 | 机会卡里有一张"直接入狱"的卡，立即入狱 |
| 🎲 移动卡连锁 | 抽到移动卡后落点在入狱格，立即入狱 |
| 🛡️ 免狱 | 以上三种场景触发前，如果手中有免狱卡，**可以此刻使用，避免入狱** |

入狱后，你的棋子移到监狱格，**当前回合立即结束**（跳过阶段④）。

### 10.2 怎么出监狱？

**只有一种方式：等待关押期满。**

- 不能支付罚金提前出狱
- 不能在入狱后使用免狱卡（免狱卡只在入狱那一刻预防使用）
- 关押期固定为 **3 轮**

出狱流程：

1. 每回合的阶段①，你的"剩余关押轮数"自动减 1
2. 当剩余关押轮数减到 0 时，**强制释放**
3. 释放后，继续本回合的后续阶段（阶段②掷骰 → ③落点 → ④动作），正常游戏

### 10.3 坐牢期间的后果

| 影响 | 说明 |
|---|---|
| 不能移动 | 你的棋子停在监狱格，跳过阶段②③ |
| 不能升级 | 监狱格不可购买或升级；你也无法踩中自己的其他地块 |
| 不能使用拆除卡 | 跳过阶段④，拆除卡无法使用 |
| 不能收租 | 你的地块在此期间**不产生租金**，别人踩上去不用付钱 |
| 回合极简 | 入狱后的回合只有阶段①（扣减轮数），然后直接结束 |

## 十一、破产——最不想看到的事

### 11.1 什么时候会破产？
当你在需要付钱（租金、卡牌扣款等）但**现金不够**时，就会触发破产处理。

### 11.2 破产处理过程（自动进行）
银行会帮你变卖资产来凑钱，规则如下：

1. 银行查看你持有的所有地块
2. 按**获取时间从早到晚**的顺序，逐个回收你的地块（先买到的先被回收）
3. 每回收一块地，银行退还你**地价（购买时支付的价格）+ 已投入的升级费总额**
   - 例如：一块地买价 $120，已升到 2 级（投入了 2 次 $50 升级费 = $100），回收价为 $120 + $100 = $220
4. 退回的钱立刻加入你的现金
5. 如果现金够了，回收停止——你勉强活下来，但失去了部分地块
6. 如果**所有地块都卖光了仍然不够**……那你真的破产了 💀
   - 如果这笔钱是要付给其他玩家的租金，地主不会收到缺少的部分，也不会收到你剩余的现金；破产后现金清零只是状态清理，不代表转给地主或银行

### 11.3 破产后
- 你退出游戏，失去所有资产
- 你持有的地块全部变回空地，任何玩家踩到都可以买
- 你持有的免狱卡和拆除卡全部作废
- 剩余玩家继续游戏

---

## 十二、游戏结束

当场上只剩下**一名未破产的玩家**时，游戏结束。

🎉 **那位玩家就是最终赢家！**

## 十三、快速参考：一页纸规则

| 你在回合里可以做什么 | 说明 |
|---|---|
| 掷骰子 | 一颗骰子，1~6，按点数走 |
| 经过起点 | 领 $200（开局位于起点不触发） |
| 踩上空地 | 阶段④买或不买，不买就没事 |
| 踩上自己的地 | 阶段④升级一级（每次踩中只能升一次） |
| 踩上别人的地 | 付租金（仅停留，路过不付） |
| 踩上机会卡格 | 抽一张卡，内容公开 |
| 踩入狱格 | 即将入狱时可用免狱卡避免；否则入狱关押 3 轮 |
| 踩上空白格 | 无事发生 |
| 使用拆除卡 | 阶段④使用，前后各 12 格内拆一块地一级 |

| 你不能做什么 | 说明 |
|---|---|
| 升级没踩中的地块 | ❌ 必须亲自走到自己的地块上才能升级 |
| 连升多级 | ❌ 每次踩中只能升一级 |
| 经过别人地块付租金 | ❌ 只有停在上面的那一下才付 |
| 提前出狱 | ❌ 只能等待 3 轮关押期满 |
| 入狱后使用免狱卡 | ❌ 只能在入狱那一刻使用，已入狱则无效 |
| 在监狱中使用拆除卡或升级 | ❌ 入狱后跳过阶段②③④ |
| 跟别人交易地块 | ❌ |
| 拍卖地块 | ❌ |
